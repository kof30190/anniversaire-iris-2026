# Brief pour l'Album de l'Anniversaire d'Iris (1 an)

Salut Claude, Hermes ici. Kof veut que tu t'occupes du design et du front-end pour l'album photo d'Iris.

## Objectif
Créer un fichier `index.html` unique (self-contained) contenant toutes les photos et les textes.

## Données
Toutes les données (images en Base64 et descriptions) sont dans le fichier `/root/anniversaire_iris/data.json`.
Le format est une liste d'objets : `[{"filename": "...", "base64": "...", "description": "..."}, ...]`.

## Contraintes de Design & Ton
- **Style** : Adapté à un premier anniversaire. Naturel, doux (thème Marais de Camargue/Candillargues), élégant mais festif.
- **Ton** : Léger, joyeux, chaleureux. Utilise les descriptions fournies et n'hésite pas à ajouter des petites transitions narratives ou des badges emojis (🍼💎🎉🎈).
- **Responsive** : Doit être parfait sur mobile (iPhone de Kof).
- **Format** : Fichier HTML UNIQUE. Tout doit être dedans (CSS Tailwind via CDN, JS éventuel pour une lightbox, images Base64).

## Structure proposée
1. **Header** : Immersif (utilise la photo de l'invitation ou une belle photo de famille).
2. **Chapitres narratifs** :
    - L'Ouverture (Invitation/Lieu)
    - La Famille (Parents, Grands-parents)
    - Le Banquet (La tablée, le repas)
    - Double Anniversaire (Iris 1 an & Jeremy 35 ans)
    - Le Gang des Tontons
    - L'Épilogue (Balade à l'étang, sieste)

## Action attendue
Génère le code du fichier `index.html` final et sauvegarde-le dans `/root/anniversaire_iris/album_iris.html`.
Donne-moi le meilleur de tes capacités en design web moderne.

Merci !
